MOT_PrintCore_iPhone_Tool v2 업로드 방법

1. GitHub 저장소의 MOT_PrintCore_iPhone_Tool 폴더 안 파일을 이번 패키지 파일로 교체합니다.
2. 특히 index.html과 service-worker.js를 반드시 함께 교체합니다.
3. 접속 주소:
   https://jeong67jea.github.io/mot-printcore.com/MOT_PrintCore_iPhone_Tool/?v=7
4. 아이폰 홈 화면에 기존 아이콘이 있으면 삭제한 뒤 Safari에서 위 주소를 열어 다시 홈 화면에 추가합니다.

Gmail·Outlook 사용
- 붙여넣기를 사용하지 않습니다.
- 내용을 입력한 뒤 약 1초 기다립니다.
- “Gmail·Outlook로 이미지 보내기”를 누릅니다.
- iOS 공유 화면에서 Gmail 또는 Outlook을 선택합니다.
- 결제안내 전체가 JPG 파일로 첨부됩니다.

네이버메일 사용
- “네이버메일 본문 복사” 후 본문에 붙여넣습니다.

중요
- Gmail·Outlook iOS 앱은 웹페이지에서 복사한 HTML/이미지를 붙여넣을 때 이미지를 제거할 수 있습니다.
- 따라서 이번 버전은 복사가 아니라 JPG 파일 공유·첨부 방식으로 해결합니다.
